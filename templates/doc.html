Authentification
POST https://woof.wannanosuurus.com/me/authWithCredential

Permet de récupérer un token d'authentification pour les accès personels woof

key value
client_id client secret id
provider le compte sur lequel se logger: google, yahoo, twitter, woof
token le token d'accèe fourni par le provider
token_secret (facultatif) token secret si fourni par le provider (ex twitter) 
curl -i -d '{"client_id" : "Qbp9mk3XgNFEu8NCCCZ06QiOvV9goTa1DLmcwmjX", "provider":"google", "token":"ya29.GluNBW5S0GMf2sL4N96Ou4EUT9khvH5jPb0Rt6qL4U3JaPwqps_n1bQdVGvaKEaYU5hNY15oqKQTOqCGksBw-vSsDFawowsrp5rcNW-7sXn-KNInEg_rXzUuiPhy"}' -H "Content-Type: application/json" -X POST https://woof.wannanosuurus.com
